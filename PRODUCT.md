# ImpactPilot AI — Product Definition

## Tagline
"Tell us what happened. We'll handle the paperwork."

## What It Is
ImpactPilot AI is a conversational documentation copilot for social impact organizations (Rotary Clubs, Rotaract Clubs, NGOs, volunteer groups). Volunteers describe an event naturally through text or voice, and the system handles all documentation automatically.

## Core Problem
After every event, volunteers must write reports, record metrics, prepare district submissions, create social media posts, and align work to SDGs. This takes 2–4 hours per event and leads to documentation fatigue, inconsistent reporting, and poor institutional memory.

## Core Solution
Two submission modes:
- **Manual Mode**: Fill a structured form.
- **AI Mode**: Type or speak naturally. AI extracts all structured data, flags missing fields, and auto-fills the form.

After submission, the system generates:
- A professional impact report
- SDG alignment (deterministic rule engine)
- Social media kit (Instagram, LinkedIn, Twitter)
- Downloadable PDF documentation package

## Target Users
- Rotary Club Presidents, Secretaries, Service Directors
- Rotaract Club Project Chairs, PR Teams, Community Service Teams
- NGOs, Student Organizations, Social Enterprises

## Key Differentiators
- Conversational capture (voice + text)
- Missing field detection with confidence indicators (✅ ⚠️ ❌)
- Deterministic SDG mapping (not probabilistic guessing)
- One-click complete documentation export
- Searchable project archive
- Leadership analytics dashboard
